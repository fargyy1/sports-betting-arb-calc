# 📰 NewsSnapper — CLI News Aggregator

Snap the latest tech news from Hacker News, Reddit, and TechCrunch — right in your terminal.

## Quick Start

```bash
cd news-snapper
python3 news_snapper.py
```

No API keys or signup required. Works on Linux, macOS, Windows (Python 3.8+).

## Features

- 📡 **3 sources**: Hacker News, Reddit r/programming, TechCrunch RSS
- ⚡ **Instant**: Fetches and displays in seconds
- 🆓 **Free**: No API keys, no accounts, no ads
- 🎨 **Clean terminal output with icons**
